使い方

クリックかキーボード入力でカウンターを操作できます。

「`」「~」で対称のカウンターを変えます。 ( )

「 [ 」「 { 」・・・ -1
「 - 」「 _ 」・・・ -10
「 ; 」「 : 」・・・ -18

「 ] 」「 } 」・・・ +1
「 = 」「 + 」・・・ +10
「 ' 」「 " 」・・・ +18

「p」でポケルスを有効にします。 ( )
「z」で一度だけ "やり直し" 機能が使えます。
